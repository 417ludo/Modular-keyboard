# Modular-keyboard
A mechanical magnetic keyboard for gaming with possibility of connect module to it 
what my keyboard have 
- it's a 60% keyboard rgb
- for gaming it has 4 magnetic button
- on the left side there is a screen
- you can attach module on the right side like a numpad and a music/ brightnesse controler

  
# What's inside
For the main keybaord 

- mechanical button
- magnetic button
- a esp32 devkitC
- magnetic sensor
- a 1.9inch screen
- 2 74hc595
- a lipo battery 5000mah
- a lipo charger
- a pogo 6pin connector
   
For the num pad

- a pogo 6 pin connector
- 19 mechanical button
- a esp32 c3 supermini
  
For the music controler

- a rotary encoder with 2 inch screen
- a esp32 c3 super mini
- a pogo 6 pin connector
